# Getting Started
_______
## Installing the Plugin

Omega Game Framework comes with 2 plugins:
* **Omega Game Framework** - the Core c++ plugin
* **Omega Demo Assets** - A vast collection of blueprints & assets utilizing OGF

Simply place these plugins in your project directory `{ProjectFolder}/Plugins` OR the engine plugins `{EngineVersion}/Engine/Plugins`.


__________
## Your First Game

After installing the plugins, your are ready to begin developing. You can start from scratch by learning about the different core features, OR you can quickly start up an example game by following these steps.

* Change Game Mode to `GM_oDemo_Gameplay`. This is and example mode for common gameplay, utilizing a Load system and an Exploration system.
* Copy the files from Plugins/OmegaGameAssets/Settings into your own project, and rename them acordingly.
* In `Project Settings > Omega: Framework` Sett all the settings to your newly created assets.
* Play your game! You now have a basic 3rd person game with full movement, jumping, sprinting, interaction, and a pause menu.
* For the most part, you will be adjusting `Gameplay Settings` and `Slate Settings`
