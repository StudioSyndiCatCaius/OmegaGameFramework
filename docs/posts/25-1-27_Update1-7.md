### Bitflags
Bitflags & Bitenums are per-project & per-class defined properties that are stored in 2 integers, thus both reducing memory cost & simplifying adding new parameters to objects.

### Global Settings/Game Core

The GameCore ("Global settings" in v1.7) is a new core GOBAL CONSTANT bluepritn class that works in both editor and runtime. It provides a long list of functions that allow you to tap into & modify tons of core Omega Game Framework functionality per-project, all in one blueprint.
